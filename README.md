# 📌 概述

**AI 算卦：** 通过6次卜筮的硬币结果生成卦象，并使用 AI 对卦象进行分析

## ⚙️ 设置

#### 环境变量

- `OPENAI_API_KEY`：不必多说，懂的都懂

## 🚀 本地运行

1. 克隆仓库：

```sh
git clone https://github.com/sunls24/divination
```

2. 安装依赖项：

```bash
pnpm install
```

3. 开发模式：

```bash
pnpm run dev
```
